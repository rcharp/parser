In order for you to take advantage of all that SimpleMetrics has to offer, we need a little bit of information from you. More specifically, we need access to your Stripe API in order to pull transaction history from your account. Use the following intstructions to get your Stripe API Credentials:



1.) Start by logging into your Stripe account at https://www.stripe.com/. Click your account name in the upper right corner to open a dropdown menu, then select "Account settings".



2.) Click the third tab in the popup box, API Settings. You will see four API keys, two for test mode and two for live mode. We need the one labeled 'Live Secret Key'. Copy and paste it into the form below, then click "Submit Credentials". Don't worry, we'll keep it safe!
